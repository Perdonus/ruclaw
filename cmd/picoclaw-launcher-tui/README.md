# RuClaw Launcher TUI

В этом каталоге находится терминальный TUI-лаунчер для `RuClaw`.
Он даёт лёгкий, нативный для терминала интерфейс для управления, настройки и работы с основным движком без веб-браузера и графической оболочки.

## Архитектура

TUI-лаунчер полностью написан на Go и не требует внешних runtime-зависимостей:
* **`main.go`**: Точка входа приложения, инициализация и основной цикл
* **`ui/`**: Компоненты TUI-интерфейса на базе `tview` + `tcell`
  - `home.go`: Главный экран и меню навигации
  - `schemes.go`: Управление схемами провайдеров моделей
  - `users.go`: Управление пользователями и API-ключами
  - `channels.go`: Редактор конфигурации каналов связи (Telegram/Discord/WeChat и т.д.)
  - `gateway.go`: Управление жизненным циклом шлюза RuClaw (старт/стоп/статус)
  - `app.go`: Базовый каркас TUI и логика навигации
  - `models.go`: Загрузка моделей и работа с состоянием
* **`config/`**: Слой управления конфигурацией, интегрированный с основной конфигурацией RuClaw

## Быстрый старт

### Требования

* Go 1.25+
* Терминал с поддержкой 256 цветов

### Разработка

Запуск TUI-лаунчера в режиме разработки:

```bash
# Из корня проекта
go run ./cmd/picoclaw-launcher-tui

# Или из этого каталога
go run .
```

### Сборка

Сборка отдельного TUI-бинаря:

```bash
# Из корня проекта (рекомендуется)
make build-launcher-tui

# Результат будет здесь:
# build/picoclaw-launcher-tui-<platform>-<arch>
# и симлинком build/picoclaw-launcher-tui

# Или собрать прямо из этого каталога
go build -o picoclaw-launcher-tui .
```

### Возможности

* 🖥️ Нативный терминальный интерфейс: работает по SSH, на headless-серверах и в лёгких окружениях
* ⚙️ Управление схемами моделей и API-ключами
* 📱 Редактор конфигурации каналов связи (Telegram/Discord/WeChat и т.д.)
* 🔄 Управление шлюзом RuClaw (старт/стоп/мониторинг статуса)
* 💬 Быстрый запуск интерактивной сессии чата
* 🎯 Управление с клавиатуры и короткие шорткаты

### Дополнительно

```bash
# Запуск с кастомным путём к конфигу
go run . /path/to/custom/config.json
```
