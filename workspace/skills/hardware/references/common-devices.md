# Справочник типовых I2C/SPI-устройств

## I2C-устройства

### AHT20 — температура и влажность
- **Адрес:** 0x38
- **Инициализация:** записать `[0xBE, 0x08, 0x00]`, подождать 10 мс
- **Измерение:** записать `[0xAC, 0x33, 0x00]`, подождать 80 мс, прочитать 6 байт
- **Парсинг:** Статус=byte[0], Humidity=(byte[1]<<12|byte[2]<<4|byte[3]>>4)/2^20*100, Temp=(byte[3]&0x0F<<16|byte[4]<<8|byte[5])/2^20*200-50
- **Примечание:** отдельной адресации регистров нет — пишите командные байты напрямую

### BME280 / BMP280 — температура, влажность, давление
- **Адрес:** 0x76 или 0x77
- **ID чипа:** регистр 0xD0 → BMP280=0x58, BME280=0x60
- **Регистры данных:** 0xF7-0xFE
- **Конфигурация:** регистры 0xF2, 0xF4, 0xF5
- **Принудительное измерение:** записать `[0x25]` в 0xF4, подождать 40 мс, прочитать 8 байт с 0xF7
- **Калибровка:** читать 26 байт с 0x88 и 7 байт с 0xE1
- **Также доступен по SPI** (mode 0 или 3)

### SSD1306 — OLED 128x64
- **Адрес:** 0x3C (или 0x3D, если SA0=high)
- **Префикс команд:** 0x00
- **Префикс данных:** 0x40
- **Команда включения:** 0xAF
- **Команда выключения:** 0xAE
- **Также доступен по SPI** — обычно быстрее для анимаций

### MPU6050 — акселерометр + гироскоп
- **Адрес:** 0x68 (или 0x69)
- **WHO_AM_I:** регистр 0x75 → должен вернуть 0x68
- **Пробуждение:** записать `[0x00]` в 0x6B
- **Акселерометр:** 6 байт с 0x3B
- **Гироскоп:** 6 байт с 0x43
- **Температура:** 2 байта с 0x41 — Temp°C = value/340 + 36.53

### DS3231 — часы реального времени
- **Адрес:** 0x68
- **Чтение времени:** 7 байт с 0x00 (BCD)
- **Установка времени:** 7 BCD-байт в 0x00
- **Температура:** 2 байта с 0x11
- **Статус:** 0x0F — bit 2 = busy, bit 0 = alarm 1 flag

### INA219 — монитор тока и мощности
- **Адрес:** 0x40-0x4F
- **Конфигурация:** регистр 0x00
- **Напряжение на шунте:** регистр 0x01
- **Напряжение шины:** регистр 0x02
- **Мощность:** регистр 0x03
- **Ток:** регистр 0x04
- **Калибровка:** регистр 0x05

### PCA9685 — 16-канальный PWM-контроллер сервоприводов
- **Адрес:** 0x40-0x7F
- **MODE1:** регистр 0x00 — bit 4=sleep, bit 5=auto-increment
- **Частота PWM:** prescale = round(25MHz / (4096 × freq)) - 1
- **Канал N:** регистры 0x06+4*N .. 0x09+4*N
- **Серво 0°-180°:** OFF=150-600 при 50 Гц

### AT24C256 — EEPROM 256 Kbit
- **Адрес:** 0x50-0x57
- **Чтение:** записать 2-байтовый адрес, затем читать N байт
- **Запись:** 2-байтовый адрес + до 64 байт данных, затем подождать около 5 мс
- **Размер страницы:** 64 байта

## SPI-устройства

### MCP3008 — 8-канальный 10-bit ADC
- **Интерфейс:** SPI mode 0, до 3.6 МГц при 5 В
- **Чтение канала N:** `[0x01, (0x80 | N<<4), 0x00]`
- **Формула:** value = ((byte[1] & 0x03) << 8) | byte[2]
- **Напряжение:** value × Vref / 1024

### W25Q128 — SPI Flash 128 Mbit
- **Интерфейс:** SPI mode 0 или 3, до 104 МГц
- **Чтение ID:** `[0x9F, 0, 0, 0]`
- **Чтение данных:** `[0x03, addr_high, addr_mid, addr_low]` + N нулевых байт
- **Статус:** `[0x05, 0]` → bit 0 = BUSY
